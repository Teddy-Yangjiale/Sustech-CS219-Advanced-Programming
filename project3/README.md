# Project 3 : High-Performance Matrix Multiplication Optimization (GEMM)

## 文件结构说明

| **文件名**                    | **说明**                                                     |
| ----------------------------- | ------------------------------------------------------------ |
| `main.c`                      | 性能测试主程序，用于测量并评估不同规模矩阵的执行时间与 GFLOPS 。 |
| `matmul.c`                    | 矩阵乘法核心逻辑演进，包含从朴素实现到寄存器阻塞微内核的多个优化版本（V1-V10） 。 |
| `matmul_arm.c`                | 针对 ARM 架构（如 Neoverse-N2）的适配实现，利用 NEON 矢量指令集进行重构 。 |
| `matmul_jit.c`                | 针对云服务器高核数环境优化的 JIT 风格版本，引入动态打包与多核并发调度技术 。 |
| `matrix.c` / `matrix.h`       | 矩阵基础数据结构定义及内存对齐分配（`aligned_alloc`）的实现 。 |
| `matrix_jit.h`                | JIT 优化版本所需的缓存配置参数与相关函数声明 。              |
| `report.pdf` / `project3.pdf` | 项目详细实验报告，记录了性能评测数据、底层架构分析及优化心得 。 |

## 简要介绍

本项目通过对矩阵乘法（GEMM）进行从算法层到指令级的深度优化，探讨了现代计算机体系结构对高性能计算的影响 。通过结合并行计算、分块缓存感知、SIMD 矢量化以及动态打包等技术，项目在 x86 与 ARM 平台上实现了高效的线性代数运算，旨在通过工程实践压榨硬件算力极限 。
